

REQUIRED SPECS TO RUN PROJECT
- XAMPP Version: 8.2.12
- JAVA java version "1.8.0_251"
- MySQL
- APACHE NETBEANS IDE 19



DB CREDENTIALS
- NAVIGATE TO SRC/ MAIN/ RESOURCES
- CREATE A config.properties FILE TO STORE YOUR DB CREDENTIALS(url, username, password, driver)
- BELOW IS A EXAMPLE OF HOW TO CREATE ONE

   please note: No spacing in config.properties file
              : NO QUATATION MARKS "" EVEN STRINGS
              : THESE ARE DEFAULT SETTINGS FOR EVERY XAMPP SERVER UNLESS CHANGED

   db.username=root
   db.password=
   db.url=jdbc:mysql://localhost:3306/vault
   db.driver=com.mysql.cj.jdbc.Driver
